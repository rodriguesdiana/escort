# Project Description

## Motivation

I had to choose a theme project to serve as an example for the students I tutor in the Algorithms and Programming Languages course in the Information Systems program.

## Objective

To become a better programmer and learn object-oriented paradigms using Python.

## What is Escort?

An escorting system where you can be either an escort or a contractor.

* **Escort:** Offers services, divided into three tiers, related to escorting.
* **Contractor:** Can hire escorts.

### Service Tiers

* Tier 1: Companion only
* Tier 2: Play only
* Tier 3: Game Night (Companion and Play)
